# Discord-bot

A discord bot with admin commands and fornite commands

## Installation

```bash
npm install
```

## Start

```bash
node .
```

## Config.Json

Please rename config.json.Template to config.json and then place in your bot token.

```bash
{
    "TOKEN": "YOURTOKENHERE"
}
```
